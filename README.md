
这是一个基于本地大模型与计算机视觉技术的微信自动化祝福工具。它能通过 **Qwen2.5-1.5B** 生成个性化文案，并利用 **PyAutoGUI 视觉识别** 精准锁定聊天目标，并模拟PC端用户操作，实现自寻找联系人/群聊以及微信祝福语生成和发送。

---

## ✨ 核心特性

* **🧠 灵魂文案**：集成 Qwen2.5，根据好友身份生成不重样的真诚祝福。
* **👁️ 视觉导航**：自研“看图识人”逻辑，通过图像识别避开搜索结果中的干扰。
* **🛡️ 安全模拟**：纯模拟真人键盘鼠标操作，不侵入内存，保障账号安全。

---

## 🚀 快速开始

1.  **环境配置**：`pip install torch transformers pywinauto pyautogui pyperclip opencv-python`。
2.  **素材准备**：截取微信搜索结果中的“群聊”和“联系人”小图标，分别保存为 `group.png` 和 `person.png`。
3.  **填写目标**：在脚本 `TARGETS` 字典中填入好友昵称及关系，在prompt中进行调整
4.  **运行脚本**：执行 `python main.py` 开启自动化祝福。
